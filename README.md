# End-to-End Housing Price Prediction and Data Science Pipeline

## Overview
This project demonstrates a complete machine learning workflow using the California Housing dataset.

The objective is to predict housing prices while showcasing modern data science techniques including:

- EDA
- Feature Engineering
- Regression
- Classification
- Clustering
- Anomaly Detection
- Deep Learning
- Explainable AI
- Model Deployment

## Dataset
California Housing Dataset from Scikit-Learn.

## Pipeline

Data Loading
→ EDA
→ Statistical Analysis
→ Feature Engineering
→ Train/Validation/Test Split
→ Machine Learning Models
→ Hyperparameter Tuning
→ Explainability
→ Deep Learning
→ Model Saving

## Models
- Linear Regression

- TensorFlow Neural Network
- PyTorch Neural Network
